# Layout Setup

#### 1. Install `Shotcut`
   - from this link → https://www.shotcut.org/download/

#### 2. Clean up Layout
   - Go to `View`  and remove tabs → `Recent, History, Keyframes`
   - Also disable from `View` → `Show Toolbar, Show Text Under Icons`
   - DO NOT DISABLE → `Show Title Bars` (because it's not possible to drag tabs without it.)

#### 3.  Make your own layout
   - Go to `View` and Enable  → `Export, Jobs`
   - Drag and drop tabs by holding → `Title Bars`.
   - Arrange all necessary Tabs as →
	   - `Playlist, Filters, Properties` -- Top left
	   - `Source, Project` -- Top right
	   - `Export, Jobs, Timeline` -- Bottom
   - Now Disable → `Show Title Bars` (to keep all tabs locked in place.)
   - Save custom layout → `View > Layout > Add`.

# Editing

#### 1. Necessary Things

   - **Add Files** → 
   - `Playlist` tab is where to bring videos, photos and everything. The `Add files to playlist` (3rd button) in `Playlist` tab is to select videos and photos.

   - **Slow Motion** →
   - select a clip and go to `properties` tab to find `speed`. More than 1x will speed up the clip and less than 1x will slow the speed of the clip.

   - **Open project** →
   - open `.mlt` file of a project to open the project.

   - **Crop a clip** → 
   - select a clip and go to `filters` tab and click the `+` plus icon and search for `Crop: source`. add it and crop the video

   - **Add Text** → 
   - select a clip and go to `filters` tab and click the `+` plus icon and search for `Text: rich` and add it. click on the text box appears above video to type and add text.

   - **Add Shape** → 
   - Go to `Open Others` and select `Animation` and `add to timeline` a box it appear. Then edit shape and just save it. Drag & drop the clip anywhere. To edit the shape again, click on the animation clip and go to `Properties` tab and click `Edit` button there.

   - **Copy filters only** → 
   - select a clip and go to `filters` tab to see the filters on it. Then select the filters and there will be a small copy icon below it to copy filters. Now select the other clip to and go to `filters` tab and there will be a small `Paste filters` button there to paste filters as it is. Filters will stay on the whole clip.

   - **Export Videos** → 
   - select a clip and go to `Export` tab, click `Advance` button from there to set export quality. click `Export Video/Audio` button to export video. The exporting progress will be shown on `Jobs` tab.

#### 2. Keyboard shortcut
   - `S` →  select a clip and press `S` from keyboard to split at video play-head.
   - `Shift + Delete` → to drag all clips from behind while deleting a clip.
   - `M` → pressing `M` will place a marker on video play-head. pressing `M` once again will let edit the marker. right click on marker to delete it.
   - `Ctrl + mouse scroll` → to shrink / expand timeline as needed.
   - `mouse scroll` → to scroll right / left in timeline.
   - `Alt + mouse scroll` → to scroll up / down in timeline.
   - `Alt + arrow keys` → to jump to the previous / next split position.
   - `Double click` → on a clip to jump the play-head at the start of the clip.
